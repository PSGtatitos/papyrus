# Papyrus — AGENTS.md

Single-file GTK4/libadwaita Python app (`papyrus.py`, 490 lines). No build system, no tests, no linter, no CI.

## Key facts

- **Entrypoint:** `papyrus.py` — shebang script installed to `~/.local/bin/papyrus`
- **Architecture:** `CWApp(Adw.Application)` with `Gio.ApplicationFlags.NON_UNIQUE`
- **Version is hardcoded** at `papyrus.py:25` — update `VERSION` string on release
- **Config:** `~/.config/papyrus/config.json` (JSON, created on first launch)
- **Thumbnails:** `~/.config/papyrus/thumbs/` — auto-generated by ffmpeg at 160px wide

## Dependencies

System: `python3-gi gir1.2-gtk-4.0 gir1.2-adw-1 ffmpeg mpvpaper`
Python: `Pillow` (installed via pip with `--break-system-packages`)

mpvpaper is mandatory — app exits immediately if not found.

## Flatpak awareness

Code checks `/app/bin/mpvpaper` exists before falling back to PATH lookup. All COSMIC theme paths are under `~/.config/cosmic/`.

## COSMIC theme integration

Writes RON-format color files to:
- `~/.config/cosmic/com.system76.CosmicTheme.{Dark,Light}/v1/{accent,background,is_dark}`
- Builder variants (`*.Builder/v1/accent`) so COSMIC Settings stays in sync

## How to run

```bash
python3 papyrus.py
# or after install:
papyrus
```

## How to install (dev iteration)

```bash
sudo apt install python3-gi gir1.2-gtk-4.0 gir1.2-adw-1 ffmpeg
pip install pillow --break-system-packages
# ensure mpvpaper is on PATH
```

No hot-reload or dev server — just re-run the script.

## Conventions

- Single-file only — do not split into modules
- No type annotations beyond what GTK introspection provides
- Video extensions: `.mp4 .webm .mkv .avi .mov` — defined in `VIDEO_EXTS` set
- Monitor output detection: `detect_outputs()` returns list of all connected monitors. Detection order: Gdk → `wlr-randr` → `["*"]`
- Per-monitor config: `wallpapers` dict maps output → path, `scaling` dict maps output → fit/fill/stretch
- Thumbnail filenames truncated to 80 chars (stem)
- **AUR package:** `papyrus-wallpaper` at `aur.archlinux.org/packages/papyrus-wallpaper`. Push to `ssh://aur@aur.archlinux.org/papyrus-wallpaper.git`. Update PKGBUILD's `pkgver` and `sha256sums` on release.
- **`loop-file=inf`** is required over `loop` — newer mpv/changed the behavior of `--loop`, so always pass `loop-file=inf` to mpv for infinite per-file looping.
